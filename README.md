# Генератор коммерческих предложений (V1)

Минимально рабочая версия: форма ввода данных о клиенте → генерация текста КП → просмотр и редактирование → скачивание PDF.

## Состав проекта (4 файла + шаблон)

- **app.py** — веб-приложение Flask: форма, генерация, просмотр/редактирование, выгрузка PDF.
- **proposal.py** — формирование текста КП по данным клиента (шаблон можно менять).
- **templates/index.html** — страница с формой и областью редактирования текста КП.
- **requirements.txt** — зависимости (Flask, reportlab).

## Запуск

1. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python app.py
   ```

4. Откройте в браузере: **http://127.0.0.1:5000**

Далее: заполните форму (ФИО, компания, контакт, тема) → нажмите «Сгенерировать КП» → при необходимости отредактируйте текст → нажмите «Скачать PDF».

## Кириллица в PDF

На Windows используется системный шрифт Arial, кириллица в PDF отображается корректно. На Linux/macOS можно положить в папку проекта файл шрифта `DejaVuSans.ttf` (например, из пакета DejaVu Fonts).
