# Генератор коммерческих предложений (V2)

Учебный проект: форма ввода данных о клиенте → выбор шаблона КП → генерация текста → просмотр и редактирование → скачивание PDF.

## Что есть в проекте

| Файл | Назначение |
|------|------------|
| **app.py** | Веб-приложение (Flask): маршруты, проверка ввода, формирование PDF. В начале файла — краткое описание структуры кода. |
| **proposal.py** | Генерация текста КП: словарь шаблонов `TEMPLATES` и функция `generate_proposal(client, template_id)`. |
| **templates/index.html** | Одна страница: форма (шаблон, ФИО, компания, контакт, тема) и блок с текстом КП + кнопка «Скачать PDF». |
| **requirements.txt** | Зависимости: Flask, reportlab. |

## Запуск

1. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python app.py
   ```

4. Откройте в браузере: **http://127.0.0.1:5000**

Дальше: выберите шаблон КП, заполните форму → «Сгенерировать КП» → при необходимости отредактируйте текст → «Скачать PDF».

## Проверки при вводе (крайние случаи)

- Все поля формы обязательны (пустые или только пробелы — ошибка).
- Длина каждого поля — не более 500 символов, длина текста КП при скачивании — не более 100 000 символов.
- Шаблон должен быть выбран из списка (иначе — ошибка).
- При скачивании PDF из текста удаляются управляющие символы, чтобы файл формировался без ошибок.

## Кириллица в PDF

На Windows используется системный шрифт Arial — кириллица в PDF отображается корректно. На Linux/macOS можно положить в папку проекта файл шрифта **DejaVuSans.ttf** (например, из пакета DejaVu Fonts).
