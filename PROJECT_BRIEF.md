# Краткое описание проекта (для вставки в начало чата)

**Генератор коммерческих предложений (КП):** форма ввода данных клиента → выбор шаблона → генерация текста КП → редактирование → скачивание PDF или сохранение в список; список сохранённых КП с поиском и просмотром.

- **app.py** — Flask-приложение: маршруты (главная, генерация, сохранение/скачивание PDF, список КП, просмотр, удаление), валидация формы, формирование PDF (reportlab), хранение КП в JSON, сортировка списка по дате.
- **proposal.py** — Генерация текста КП: словарь шаблонов `TEMPLATES` и функция `generate_proposal(client, template_id)`.
- **templates/index.html** — Главная страница: форма (клиент + шаблон), блок с текстом КП, кнопки «Скачать PDF» и «Сохранить в список».
- **templates/list.html** — Список сохранённых КП (поиск, сортировка по дате), ссылки на просмотр и удаление.
- **templates/view.html** — Просмотр одного сохранённого КП и кнопка «Скачать PDF».
- **requirements.txt** — Зависимости: Flask, reportlab.
- **README.md** — Документация и инструкция по запуску.
- **.gitignore** — Исключения для Git (venv, saved_proposals.json и др.).

Данные сохранённых КП хранятся в **saved_proposals.json** (создаётся при первом сохранении, в .gitignore).
